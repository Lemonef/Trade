Construction

Universe (PIT)Survivorship-bias-free: listing-date gating (+30d), rolling 30d median $ADV ≥ ~$20M, delisting exit at last price, symbol-migration mapping, perp/funding availability for the leveraged version. Eligibility rebuilt weekly.
SignalRank eligible coins by trend strength (Donchian 55/20 breakout + normalized momentum). Hold top 8, equal-weight. Evaluate on the 4H close. Rank-hysteresis buffer (swap only if a challenger beats a held coin by ≥2 ranks) to cut churn.
Regime gateBTC > MA200 = risk-on (the validated master filter that halved 2022 losses). Risk-off → cash, no new entries.
SizingATR risk sizing (~0.5–1%/trade); funding-aware throttle (cut size when perp funding hot — cost/risk, not alpha); leverage ≤1.5x core / ≤2x high-CAGR; per-coin ≤15%, capped if top-N highly correlated.
Risk / roleDonchian-20 / ATR stop; max-DD guardrail de-grosses on breach. v1 = core role 1–1.5x. Keep the existing Donchian core running as the benchmark.
